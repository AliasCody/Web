<span style='font-size:24px;'>Everythings here start from</span>

```bash
python manage.py startapp core
```

<span style='font-size:24px;'>So the beginning structure will be like this ( after create `config/` )</span>

```bash
Personal_Web/
│
├─ manage.py
├─ config/        ← Project
│
└─ core/          ← App
   ├─ views.py
   ├─ models.py
   ├─ admin.py
   ├─ apps.py
   ├─ tests.py
   └─ migrations/
```
